# Task-03-Prachi-AI_Recommendation_Logic

## Project Description

The AI Recommendation System is a simple Python-based application that recommends movies, books, and music based on the user's interests. The program accepts a category and an interest as input, matches them with predefined data, and displays suitable recommendations using basic AI recommendation logic.

## Features

- User-friendly interface
- Movie recommendations
- Book recommendations
- Music recommendations
- Input validation
- Continuous recommendation loop
- Exit option

## Technologies Used

- Python
- Dictionary
- Lists
- if-else statements
- for loop
- while loop
